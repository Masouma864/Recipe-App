## Getting Started
To get a local copy up and running follow these simple example steps.

- Clone this repo using this URL (link) or download it as a ZIP format from the green button above under the text Code.

### Install
```shell
npm install
```

### Run tests
```shell
npm run test
```

### Deployment
```shell
npm run deploy
```
